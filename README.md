# agent-skills

Security and productivity plugins for Claude Code.

## Installation

Add the marketplace:

```
/plugin marketplace add mfakbar127/agent-skills
```

Install the security plugin:

```
/plugin install security-code-audit@agent-skills
```

## Available Plugins

### security-code-audit

Security-focused code review toolkit using grepai semantic search. Provides vulnerability scanning, data flow analysis, and security Q&A capabilities.
